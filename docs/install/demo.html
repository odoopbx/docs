The easiest and fastest way to start using OdooPBX is the docker based installation. Full demo is available for direct setup only with Asterisk (direct setup does not install FreePBX currently).

Run either Asterisk or FrePBX service.

docker-compose up odoo agent freepbx

See Docker based deployment for details.
