Res Users¶

ResUser.asterisk_plus_notify(message, title='PBX', uid=None, sticky=False, warning=False)¶

Send a notification to logged in Odoo user.

Args:

message (str): Notification message. title (str): Notification title. If not specified: PBX. uid (int): Odoo user UID to send notification to. If not specified: calling user UID. sticky (boolean): Make a notiication message sticky (shown until closed). Default: False. warning (boolean): Make a warning notification type. Default: False.

Returns:

Always True.
