================================
Zoiper installation instructions
================================
Make sure you know
  * **Hostname** of SIP server (``sip.example.com`` is used here )
  * **Username** (``701`` is used here)
  * **Password**


Zoiper for Android
==================
Install Zoiper IAX SIP VOIP Softphone from
`Google Play <https://play.google.com/store/apps/details?id=com.zoiper.android.app>`_.
Run it. You will be asked to agree to Terms & Conditions and permissions to make phone
calls and run in the background. We suggest that you accept and allow all.

You are prompted to setup an account. Enter your credentials like below:

.. image:: media/zoiper_account_setup_0.jpg
   :alt: Initial screen
   :width: 300 px

.. image:: media/zoiper_account_setup_1.jpg
   :alt: Enter username and password
   :width: 300 px

And press *Create account*.

At the next screen fill in your **hostname** (it may be already preset) and press *Next*.

At the next screen select "*My provider/PBX requires an authentication or outbound proxy*"
checkbox and fill in your **username** and **hostname** again. See below an example:

.. image:: media/zoiper_account_setup_2.jpg
   :alt: Enter username and password
   :width: 300 px

Press Next. Choose available protocol (marked with green color).

.. image:: media/zoiper_account_setup_3.jpg
   :alt: Choose available protocol
   :width: 300 px

Press Next. Thats all. Now you can dial any number by invoking dialpad from middle-top icon:

.. image:: media/zoiper_account_setup_4.jpg
   :alt: Dial any number
   :width: 300 px


Zoiper Desktop
==============
Install `Zoiper Desktop <https://www.zoiper.com/en/voip-softphone/download/current>`_.
Run it. Choose *Continue as a free user*.

.. image:: media/zoiper_desktop_1.png
   :alt: Zoiper Desktop 1

Fill in your **username@hostname** and **password**. For example:

.. image:: media/zoiper_desktop_2.png
   :alt: Zoiper Desktop 2

At the next screen fill in your **hostname** (it may be already preset) and press *Next*.

.. image:: media/zoiper_desktop_3.png
   :alt: Zoiper Desktop 3

At the next screen select *Authentication and Outbound proxy* checkbox and fill in your
**username** and **hostname** again. Look how we did:

.. image:: media/zoiper_desktop_4.png
   :alt: Zoiper Desktop 4

Choose available protocol (marked with green color).

.. image:: media/zoiper_desktop_5.png
   :alt: Zoiper Desktop 5

That's it. Now you can enter destination number in search box and press dial button.

.. image:: media/zoiper_desktop_6.png
   :alt: Zoiper Desktop 6
